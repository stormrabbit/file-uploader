# 行为准则 Code of Conduct

## 我们的承诺

作为成员、贡献者和维护者，我们承诺为每一位参与者提供友好、安全和热情的环境，无论其年龄、体型、残疾状况、民族、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向如何。

## 我们的标准

**有助于营造积极环境的行为包括：**

- 使用友好、包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

**不可接受的行为包括：**

- 使用带有性暗示的语言或图像，以及不受欢迎的性骚扰
- 恶意评论、人身攻击或政治攻击（Trolling / Insulting）
- 公开或私下骚扰
- 未经明确许可，发布他人的私人信息（如实际或电子地址）
- 其他在专业场合被合理认为不当的行为

## 维护者的责任

项目维护者有责任明确可接受行为的标准，并对任何不可接受的行为采取适当且公正的纠正措施。

维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、Issues 及其他贡献，并可临时或永久封禁行为不当的贡献者。

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公共场合代表本项目或其社区时。代表本项目的行为包括：使用官方项目电子邮件地址、通过官方社交媒体账号发帖，或在线上/线下活动中担任指定代表。

## 执行

如发现辱骂、骚扰或其他不可接受的行为，可通过项目 Issues 向维护者举报。所有投诉都将被审查和调查，并作出必要且适当的回应。维护者有义务对事件报告者保密。
